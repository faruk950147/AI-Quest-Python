Django Built-in Authentication System – Full Notes
১. Authentication URLs include করা

Django-এর built-in authentication system ব্যবহারের জন্য প্রথমে urls.py-তে নিচের কোডটি লিখতে হবে:

from django.urls import path, include

urlpatterns = [
    path('accounts/', include('django.contrib.auth.urls')),
]

ব্যাখ্যা:

accounts/ হলো base URL।

এটি automatically নিচের URLগুলো যোগ করে দেয়:

URL	Purpose
/accounts/login/	Login page
/accounts/logout/	Logout page
/accounts/password_change/	Change password form
/accounts/password_change/done/	Password changed successfully
/accounts/password_reset/	Password reset request form
/accounts/password_reset/done/	Password reset email sent
/accounts/reset/<uidb64>/<token>/	Password reset confirmation
/accounts/reset/done/	Password reset complete
২. Templates তৈরি করা

Django built-in authentication system templates খুঁজে পায় registration/ folder থেকে। তাই project এর templates folder এর মধ্যে তৈরি করতে হবে:

৩. Login / Logout Behavior

Login: /accounts/login/

ডিফল্ট form: username + password

Successful login → redirect to /accounts/profile/ (ডিফল্ট, পরিবর্তন করতে হবে settings.py এ)

# settings.py
LOGIN_REDIRECT_URL = '/'  # login successful হলে redirect হবে এই URL-এ
LOGOUT_REDIRECT_URL = '/'  # logout successful হলে redirect হবে

Logout: /accounts/logout/

Logged out হওয়ার পর redirect হবে logged_out.html বা LOGOUT_REDIRECT_URL-এ

৪. Password Change & Reset
Password Change

/accounts/password_change/ → old password দিয়ে change করা

/accounts/password_change/done/ → change complete message

Password Reset (Email Required)

/accounts/password_reset/ → email দিয়ে password reset request

/accounts/password_reset/done/ → email sent confirmation

/accounts/reset/<uidb64>/<token>/ → reset link click করে new password

/accounts/reset/done/ → reset complete

Email Setup (settings.py):

EMAIL_BACKEND = 'django.core.mail.backends.console.EmailBackend'  # development
# production এ SMTP ব্যবহার করতে হবে
৫. Customization

Custom Login URL

# urls.py
path('user/', include('django.contrib.auth.urls')),  # Login now: /user/login/

Custom Templates

শুধু template file বানাতে হবে registration/ folder-এ। Django automatically detect করবে।

Redirect URLs

# settings.py
LOGIN_REDIRECT_URL = '/dashboard/'
LOGOUT_REDIRECT_URL = '/goodbye/'

Form Customization

{{ form.as_p }} এর জায়গায় field গুলো নিজে control করতে পারো:

<input type="text" name="username" placeholder="Username">
<input type="password" name="password" placeholder="Password">
What is this?
৬. Example Workflow

User visits /accounts/login/ → login form দেখাবে

User submits credentials → valid হলে redirect হবে LOGIN_REDIRECT_URL এ

User wants to change password → /accounts/password_change/

User forgets password → /accounts/password_reset/ → email link → set new password

৭. Tips

Always add {% csrf_token %} in form

Login template এ errors দেখানোর জন্য:

{% if form.errors %}
  <p style="color:red;">Invalid username or password</p>
{% endif %}
What is this?

Production environment এ email backend properly configure করতে হবে password reset এর জন্য।
