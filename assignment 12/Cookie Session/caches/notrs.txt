🔥 1. Caching কী?

Caching মানে হচ্ছে —
👉 বারবার calculate না করে data আগেই save করে রাখা

📌 Example:

Database query slow
API call slow
Template render slow

➡️ এগুলো cache করলে next time fast load হবে

⚡ 2. Django-তে caching কেন ব্যবহার করবো?

✅ Website fast হয়
✅ Database load কমে
✅ Server cost কমে
✅ User experience better হয়

🧠 3. Django caching types

Django-তে 3 ধরনের caching বেশি use হয়:

1. Per-site cache

👉 পুরো website cache

2. Per-view cache

👉 নির্দিষ্ট view cache

3. Template fragment cache

👉 template-এর কিছু অংশ cache

🏗️ 4. Django cache backend

Django cache store করার জন্য backend লাগে:

🔹 Built-in backends:
LocMemCache (local memory)
FileBasedCache
DatabaseCache
🔹 Production backend:
Redis ✅ (best)
Memcached
⚙️ 5. Basic setup (settings.py)
🟢 Simple local cache:
CACHES = {
    "default": {
        "BACKEND": "django.core.cache.backends.locmem.LocMemCache",
        "LOCATION": "unique-snowflake",
    }
}
🚀 6. Per-view cache
Example:

@cache_page(60 * 5)  # 5 minutes

    ...

🧪 9. Low-level cache API (MOST IMPORTANT)

👉 Advanced use (dynamic control)

from django.core.cache import cache

# Set
cache.set('my_key', 'hello', timeout=60)

# Get
data = cache.get('my_key')

# Delete
cache.delete('my_key')

# Clear all
cache.clear()
🎯 10. Cache key concept

👉 Cache key unique identifier

cache.set(f"product_{id}", product_data)
🔁 11. Cache invalidation (VERY IMPORTANT)

👉 Data change হলে cache delete করতে হবে

cache.delete(f"product_{id}")

📌 না করলে old data দেখাবে

⚡ 12. Redis cache (Production level)

👉 Best performance

Install:
pip install django-redis
settings.py:
CACHES = {
    "default": {
        "BACKEND": "django_redis.cache.RedisCache",
        "LOCATION": "redis://127.0.0.1:6379/1",
        "OPTIONS": {
            "CLIENT_CLASS": "django_redis.client.DefaultClient",
        }
    }
}
🔥 13. Real-life example (E-commerce)
Product page cache:
def product_detail(request, id):
    cache_key = f"product_{id}"
    product = cache.get(cache_key)

    if not product:
        product = Product.objects.get(id=id)
        cache.set(cache_key, product, 300)

    return render(request, "product.html", {"product": product})
