Django-তে WebSocket ব্যবহার

Django সরাসরি WebSocket support করে না। এজন্য ব্যবহার করা হয়:

Django Channels

এটি Django-কে asynchronous এবং WebSocket compatible করে।

Basic Setup
1. Install
pip install channels
2. settings.py
INSTALLED_APPS = [
    'channels',
]

ASGI_APPLICATION = 'project_name.asgi.application'
3. asgi.py
import os
from channels.routing import ProtocolTypeRouter, URLRouter
from django.core.asgi import get_asgi_application
from channels.auth import AuthMiddlewareStack
import app.routing

os.environ.setdefault('DJANGO_SETTINGS_MODULE', 'project_name.settings')

application = ProtocolTypeRouter({
    "http": get_asgi_application(),
    "websocket": AuthMiddlewareStack(
        URLRouter(
            app.routing.websocket_urlpatterns
        )
    ),
})
4. routing.py
from django.urls import path
from . import consumers

websocket_urlpatterns = [
    path('ws/chat/', consumers.ChatConsumer.as_asgi()),
]
5. Consumer (WebSocket Logic)
from channels.generic.websocket import WebsocketConsumer
import json

class ChatConsumer(WebsocketConsumer):

    def connect(self):
        self.accept()

    def disconnect(self, close_code):
        pass

    def receive(self, text_data):
        data = json.loads(text_data)
        message = data['message']

        self.send(text_data=json.dumps({
            'message': message
        }))
6. Frontend JavaScript
const socket = new WebSocket('ws://127.0.0.1:8000/ws/chat/');

socket.onmessage = function(e) {
    const data = JSON.parse(e.data);
    console.log(data.message);
};

socket.onopen = function() {
    socket.send(JSON.stringify({
        'message': 'Hello Server'
    }));
};
কাজের ধাপ
Client WebSocket connect করে
Server connection accept করে
Client message পাঠায়
Server সাথে সাথে response দেয়
Page reload প্রয়োজন হয় না
HTTP vs WebSocket
বিষয়	HTTP	WebSocket
Connection	Short-lived	Persistent
Communication	One-way (request-response)	Two-way
Real-time	না	হ্যাঁ
Use case	API	Chat, notification
