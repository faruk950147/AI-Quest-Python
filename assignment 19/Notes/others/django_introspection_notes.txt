Example 1: Generic Function (Multi Model Support)

ধরো তোমার 2টা model আছে:

class Product(models.Model):
    name = models.CharField(max_length=100)

class Customer(models.Model):
    email = models.EmailField()

def show_info(obj):
    print("App:", obj._meta.app_label)
    print("Model:", obj._meta.model_name)
    print("Class Name:", obj._meta.object_name)
    
Use:
product = Product.objects.get(id=1)
customer = Customer.objects.get(id=1)

show_info(product)
show_info(customer)
Output:
App: shop
Model: product
Class Name: Product

App: shop
Model: customer
Class Name: Customer
Example 2: Dynamic JSON API (Very Important)
from django.http import JsonResponse

def get_data(request):
    product = Product.objects.get(id=1)

    data = {
        "model": product._meta.model_name,
        "app": product._meta.app_label,
        "name": product.name,
    }

    return JsonResponse(data)
Output JSON:
{
  "model": "product",
  "app": "shop",
  "name": "Laptop"
}
