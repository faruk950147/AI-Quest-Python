Django + Gunicorn Setup on WSL (Windows Subsystem for Linux)
1️ WSL Environment Preparation
WSL Open:
Windows এ WSL Terminal open করো (WSL 2 recommended)।
Check Python & pip:
python3 --version
pip3 --version
Project Directory এ যাও:
cd /mnt/c/Users/Faruk/Desktop/myproject
Virtual Environment তৈরি ও Activate:
python3 -m venv venv
source venv/bin/activate
Required Packages Install:
pip install django gunicorn
2️ Gunicorn Test Run
Gunicorn দিয়ে run করো:
gunicorn myproject.wsgi:application --bind 127.0.0.1:8000 --workers 3
Check in Browser:
Windows browser এ খুলে দেখো: http://localhost:8000
যদি সব ঠিক থাকে, তাহলে project live।
3️ Background এ Gunicorn Run
WSL 2 systemd support না দিলে terminal close হলে stop হবে।
Background run করতে:
nohup gunicorn myproject.wsgi:application --bind 127.0.0.1:8000 --workers 3 &
& → background
nohup → terminal বন্ধ হলেও run করবে
Running processes check:
ps aux | grep gunicorn
Stop process:
kill <PID>
4️ Static Files Setup
Django static files collect:
python manage.py collectstatic
Development mode এ static serve করার জন্য:
python manage.py runserver
Dev purpose এ NGINX দরকার নেই।
5️ Optional: NGINX Setup on WSL (Advanced)
Install NGINX:
sudo apt update
sudo apt install nginx
Config File:
server {
    listen 80;
    server_name localhost;

    location /static/ {
        root /mnt/c/Users/Faruk/Desktop/myproject;
    }

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
Enable & Restart:
sudo ln -s /etc/nginx/sites-available/myproject /etc/nginx/sites-enabled
sudo nginx -t
sudo systemctl restart nginx

Note: WSL firewall ও network interface handle করতে হবে।

6️ Tips for WSL Usage
WSL mainly development/test purpose এর জন্য ভালো।
Production deploy করতে Linux VM / Cloud server use করা ভালো।
Background run করতে nohup + & combination ব্যবহার করতে হবে।
Browser access করতে localhost:8000 বা 127.0.0.1:8000 use করো।
