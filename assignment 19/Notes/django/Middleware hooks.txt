Django-তে Middleware hooks হলো এমন কিছু special methods যেগুলো request এবং response lifecycle-এর বিভিন্ন stage-এ run হয়। অর্থাৎ browser → server → view → response এই flow-এর মাঝে middleware code execute করা যায়।

সহজভাবে বললে:
Middleware = request এবং response এর মাঝখানে কাজ করা layer.

Django Middleware Hooks

Django middleware-এ সাধারণত কয়েকটি hook method ব্যবহার করা হয়।

1️⃣ __init__(self, get_response)

Middleware প্রথমবার load হলে এই method run হয়।

class MyMiddleware:
    def __init__(self, get_response):
        self.get_response = get_response

⚠️ এটা server start হলে একবারই run হয়।

2️⃣ __call__(self, request)

প্রতিটি request এ এই method run হয়।

class MyMiddleware:
    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):
        # request এর আগে code
        print("Before view")

        response = self.get_response(request)

        # response এর পরে code
        print("After view")

        return response

3️⃣ Rate Limiting (API Security)

একজন user বেশি request করলে block করা।

import time

class RateLimitMiddleware:

    last_request = {}

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        ip = request.META.get("REMOTE_ADDR")
        now = time.time()

        if ip in self.last_request and now - self.last_request[ip] < 1:
            from django.http import HttpResponse
            return HttpResponse("Too many requests")

        self.last_request[ip] = now

        return self.get_response(request)

1️⃣ JWT Authentication Middleware

এই middleware Authorization header থেকে JWT token check করে user authenticate করতে পারে।

import jwt
from django.conf import settings
from django.http import JsonResponse
from django.contrib.auth import get_user_model

User = get_user_model()

class JWTAuthMiddleware:

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        token = request.headers.get("Authorization")

        if token:
            token = token.replace("Bearer ", "")

            payload = jwt.decode(token, settings.SECRET_KEY, algorithms=["HS256"])

            user_id = payload.get("user_id")

            user = User.objects.filter(id=user_id).first()

            request.user = user

        return self.get_response(request)

3️⃣ Role Permission Middleware

User role অনুযায়ী page access control করা।

from django.http import HttpResponseForbidden

class RoleMiddleware:

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        if request.path.startswith("/admin-panel"):

            if not request.user.is_authenticated:
                return HttpResponseForbidden("Login required")

            if request.user.role != "admin":
                return HttpResponseForbidden("Access denied")

        return self.get_response(request)

5️⃣ API Rate Limiting Middleware

একজন user বেশি request করলে block করা।

import time
from django.http import JsonResponse

class APIRateLimitMiddleware:

    requests = {}

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        ip = request.META.get("REMOTE_ADDR")
        now = time.time()

        if ip in self.requests:

            if now - self.requests[ip] < 1:
                return JsonResponse({"error": "Too many requests"}, status=429)

        self.requests[ip] = now

        return self.get_response(request)

1️⃣ Middleware vs Decorator vs Signals
Feature	Middleware	Decorator	Signals
কোথায় কাজ করে	Global (সব request)	Specific view	Event based
কখন run হয়	Request/Response cycle	View call এর আগে/পরে	নির্দিষ্ট event হলে
Scope	পুরো project	নির্দিষ্ট view	Model/Framework events
Example	Authentication	Login required	Post save

Decorator Example

শুধু নির্দিষ্ট view-এ কাজ করে।

from django.http import HttpResponse

def my_decorator(view_func):

    def wrapper(request, *args, **kwargs):
        print("Before view")
        return view_func(request, *args, **kwargs)

    return wrapper
@my_decorator
def home(request):

2️⃣ Production Level Logging Middleware

Production project এ request log + response time track করা হয়।

import time
import logging

logger = logging.getLogger(__name__)

class LoggingMiddleware:

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        start = time.time()

        response = self.get_response(request)

        duration = time.time() - start

        logger.info(
            f"{request.method} {request.path} {response.status_code} {duration:.2f}s"
        )

        return response

4️⃣ Custom Security Middleware

class SecurityHeadersMiddleware:

    def __init__(self, get_response):
        self.get_response = get_response

    def __call__(self, request):

        response = self.get_response(request)

        response["X-Frame-Options"] = "DENY"
        response["X-Content-Type-Options"] = "nosniff"
        response["X-XSS-Protection"] = "1; mode=block"

        return response
