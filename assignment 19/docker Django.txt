Django + Docker All Commands (Complete Guide)
1. Docker Version Check
docker --version
docker compose version

2. Docker Image Build
docker build -t django-app .

এখানে django-app হলো image name

3. Docker Container Run
docker run -p 8000:8000 django-app

ব্রাউজার:

http://localhost:8000
4. Background (Detached Mode)
docker run -d -p 8000:8000 django-app

5. Running Containers List
docker ps

All container (stopped সহ)

docker ps -a
6. Stop Container
docker stop container_id

7. Remove Container
docker rm container_id

8. Remove Image
docker rmi image_id

9. Docker Logs (Error দেখার জন্য)
docker logs container_id

Live logs:

docker logs -f container_id
Docker Compose (Django + MySQL)

modern Django project এ এটা সবচেয়ে বেশি ব্যবহার হয়

10. Start Services
docker compose up

Background:

docker compose up -d
11. Build + Start
docker compose up --build

12. Stop Services
docker compose down

13. Restart Services
docker compose restart

14. View Running Services
docker compose ps

15. Logs (Compose)
docker compose logs

Live logs:

docker compose logs -f
16. Execute inside Django Container
docker exec -it container_id bash

এরপর Django commands চালাতে পারো:

python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
Django Common Docker Commands
Migrations
python manage.py makemigrations
python manage.py migrate
Superuser
python manage.py createsuperuser
Docker Clean Commands
Remove all stopped containers
docker container prune
Remove unused images
docker image prune
Full system clean
docker system prune -a
