Django + Docker All Commands (Complete Guide)
1. Docker Version Check
docker --version
docker compose version

2. Docker Image Build
docker build -t django-app .

================ Image Name =================
here django-app is image name

3. Docker Container Run
docker run -p 8000:8000 django-app

================ Browser =================

http://localhost:8000
4. Background (Detached Mode)
docker run -d -p 8000:8000 django-app

5. Running Containers List
docker ps

All container (stopped)

docker ps -a
6. Stop Container
docker stop container_id

7. Remove Container
docker rm container_id

8. Remove Image
docker rmi image_id

9. Docker Logs (Show Error)
docker logs container_id

Live logs:

docker logs -f container_id
